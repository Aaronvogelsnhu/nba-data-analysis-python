# NBA Data Analysis – Python

## Overview
This project focuses on analyzing NBA team data using Python to identify performance trends and statistical patterns. The goal was to take raw data and turn it into meaningful insights using code and visualizations.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## What I Did
- Loaded and cleaned NBA dataset
- Calculated descriptive statistics and performance metrics
- Created visualizations to analyze team performance
- Identified patterns in wins, scoring, and efficiency

## Key Skills Demonstrated
- Data analysis with Python
- Data cleaning and transformation
- Statistical analysis
- Data visualization
- Working with real-world datasets

## Outcome
This project improved my ability to analyze data using Python and extract meaningful insights from large datasets. It also strengthened my understanding of how data can be used to support decision-making.
